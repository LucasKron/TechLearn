O presente projeto tem como objetivo solucionar o déficit de conhecimentos
tecnológicos, tais conhecimentos se tornaram indispensáveis com o avanço
tecnológico, tanto para vida cotidiana quanto para o mercado de trabalho. Dessa
forma, o desenvolvimento do TechLearn, um projeto que pode ser descrito como uma
abordagem educacional web, tem como finalidade preparar mais pessoas com
conhecimentos essenciais de tecnologia para a realização de tarefas do dia a dia, com
ênfase âmbito profissional para entrada no mercado de trabalho.
